#### 3.1 用户登录模块（3天）⭐ **先学这个！**

> 很多功能需要登录，所以先学登录！

**第1步：理解登录流程（1天）**

```
1. 用户输入账号密码
   ↓
2. 点击登录按钮
   ↓
3. 调用登录 API
   ↓
4. 服务器返回 token 和用户信息
   ↓
5. 保存到 Pinia Store 和 localStorage
   ↓
6. 跳转到首页（或原本要去的页面）
```

**第2步：阅读代码（1天）**

```vue
<!-- src/views/user/login.vue -->
<script setup>
import { useUserStore } from '@/stores/user'
import { useRouter, useRoute } from 'vue-router'

const userStore = useUserStore()
const router = useRouter()
const route = useRoute()

const handleLogin = async () => {
  // 1. 调用 Store 的 login 方法
  await userStore.login({ username, password })
  
  // 2. 跳转回原本要去的页面
  const redirect = route.query.redirect || '/home'
  router.push(redirect)
}
</script>
```

**第3步：理解登录拦截（1天）**

```javascript
// src/router/index.js
router.beforeEach((to, from, next) => {
  if (to.meta.requireAuth) {  // 需要登录的页面
    const userStore = useUserStore()
    if (!userStore.isLogin()) {  // 未登录
      next('/login?redirect=' + to.fullPath)  // 跳转登录页
      return
    }
  }
  next()  // 放行
})
```

**第2步：核心功能实现（1.5天）**

**功能1：全选/取消全选**
```javascript
const allChecked = computed(() => {
  return cartList.value.every(item => item.checked)
})

const toggleAll = () => {
  const checked = !allChecked.value
  cartList.value.forEach(item => {
    item.checked = checked
  })
}
```

**功能2：计算总价**
```javascript
const total = computed(() => {
  return cartList.value
    .filter(item => item.checked)
    .reduce((sum, item) => {
      return sum + item.product.price * item.quantity
    }, 0)
})
```

**功能3：修改数量**
```javascript
const updateQuantity = async (item, quantity) => {
  await cartStore.updateCart({
    id: item.id,
```

#### 4.2 性能优化技巧（2天）

**优化1：路由懒加载（已实现）**
```javascript
// ❌ 不好：一次性加载所有页面
import HomePage from '@/views/home/index.vue'

// ✅ 好：访问时才加载
component: () => import('@/views/home/index.vue')
```

**优化2：图片懒加载**
```vue
<!-- 使用 Vant 的 lazy-load -->
<van-image lazy-load :src="imageUrl" />
```

**优化3：防抖（debounce）**
```javascript
// 搜索框输入时，停止输入 300ms 后才搜索
import { debounce } from 'lodash-es'

const handleSearch = debounce((keyword) => {
  searchProducts(keyword)
}, 300)
```

**优化4：节流（throttle）**
```javascript
// 滚动加载时，每 200ms 最多触发一次
import { throttle } from 'lodash-es'

const handleScroll = throttle(() => {
  loadMore()
}, 200)
```

1. 能解释防抖和节流的区别

### 3. 对比学习 ⚖️

**理解"为什么这样做"：**

|------|---------------|-------------|---------|

## ❓ 常见问题解答

### Q1: 我完全是新手，8周能学完吗？
**A:** 8周是理想时间，实际时间因人而异：
- 如果有 JavaScript 基础：6-8周
- 如果是纯新手：需要 10-12周（包括学习 JS 基础）
- **重要：** 不要着急，理解比速度更重要！

### Q2: 学习过程中卡住了怎么办？
**A:** 三步解决法：
1. **查文档** - 80% 的问题文档都有答案
2. **搜索** - Google/百度搜索错误信息
3. **提问** - 在社区提问（附上代码和错误信息）

### Q3: Vue 2 和 Vue 3 有什么区别？
**A:** 主要区别：
- Vue 3 使用 Composition API（`<script setup>`）
- Vue 3 性能更好，体积更小
- **建议：** 新项目直接学 Vue 3

### Q4: 需要先学会 Vue 2 吗？
**A:** **不需要！** 直接学 Vue 3 即可，它更现代、更简洁。

### Q5: 学完这个项目能找工作吗？
**A:** 这个项目是很好的练习，但找工作还需要：
- 深入理解原理（不只是会用）
- 做 2-3 个不同类型的项目
- 学习计算机基础知识（算法、网络等）
- 准备面试题

### Q6: 每天学多久合适？
**A:** 建议：
- **工作日：** 2-3 小时
- **周末：** 4-6 小时
- **重要：** 保持持续，不要间断太久

### Q7: 需要背代码吗？
**A:** **不需要背！** 理解逻辑更重要：
- 理解"为什么这样写"
- 理解"数据如何流转"
- 用的多了自然就记住了
